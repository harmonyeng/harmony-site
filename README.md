# Harmony Engineering: The Home Operations System (HOS)

## Executive Summary
Harmony Engineering is a pioneering "Steel Silk" platform designed to professionalize home management. By applying Industrial Engineering (IE) principles to domestic operations, we transform household chaos into high-performance efficiency. This system empowers the "CEO of the Home" to move from survival mode to strategic engineering.

## Key Features
* **Steel Silk Interface:** A premium, minimalist UI featuring a proprietary "Vertical Split Reveal" and a haptic 3D Command Center.
* **Standard Operating Procedures (SOPs):** Engineering-grade protocols for domestic tasks, focusing on flow and efficiency rather than "perfection."
* **The Ops Center:** A strategic Q&A hub for data-driven household decision-making.
* **Amazon My Love Boutique:** Integrated curated gallery for premium publications and IE-inspired productivity tools.
* **Brave Man Access:** A technical entry point designed to onboard male household managers into the professionalized ecosystem.

## Prerequisites
* **Hosting:** Vercel account for automated deployment.
* **Source Control:** GitHub account (User: `harmonyeng`).
* **Environment:** Node.js 14+ environment.
* **Admin:** Configured via `harmonyengineeringtn@gmail.com`.

## Operating Instructions
1. **Clone Repository:** `git clone https://github.com/harmonyeng/harmony-site.git`
2. **Setup Env:** Create `.env.local` and define `ADMIN_EMAIL`.
3. **Install:** Run `npm install` to sync dependencies (Tailwind, NextAuth).
4. **Deploy:** Push to `main` branch for automated Vercel live-sync.
